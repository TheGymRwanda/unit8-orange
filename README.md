## Getting Started

### Prerequisites

- nodejs <br />
  [Node install](https://nodejs.org/en/download/)

### Installation

1. clone the project<br/>

```bash
git clone https://github.com/TheGymRwanda/unit8-orange.git
```

2. install packages

```bash
npm install
```

3. start development server

```bash
npm run dev
```

4. open the localhost url in the browser
